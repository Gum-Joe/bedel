Getting started

Before you get started, check you have the following:

  • node.js >= 6.0.0
  • mongodb >= 2.0.0

As we are yet to tag our first release, clone this repo and run:

cd path/to/bedel
npm install
To start bedel:
bin/bedel

Note: You may need to add a username and password manually to you mongodb in the 'bedel' database. Please use bcryptjs to hash the password with a salt factor of 10.
